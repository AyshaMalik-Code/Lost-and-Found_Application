# Lost & Found Management System

## Overview

The Lost & Found Management System is a web-based application that helps users report, manage, and track lost and found items. The platform provides a centralized system where users can post information about lost belongings and found items, making it easier to reconnect owners with their possessions.

---

## Features

- User-friendly interface
- Add lost item details
- Add found item details
- Upload item images
- View all reported items
- Search and manage records
- MongoDB database integration
- RESTful API implementation
- Responsive design

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/lost-and-found-system.git
cd lost-and-found-system
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

## Run the Project

### Start Backend Server

```bash
cd backend
npm start
```

### Start Frontend

Open a new terminal and run:

```bash
cd frontend
npm start
```
